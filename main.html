PASSPAL
github.com/nickryt/passpal
This application was my submission for the Rutgers' 24-Hour "HackRU"" Hackathon in Fall 2020. In this short time span, I created a fully-functional password manager using Python and Tkinter. My goal with this project was to create a personal and 100% offline password manager - I do not trust other password managers with my information! I used Fernet encryption in this project, allowing users to save their password data files without their information being saved in plaintext. Users need to know their key or pass in a key file to decode their information. While "HackRU" has ended, I still maintain this code and plan on implementing more functionality (such as storing credit card information and more) in the future.
